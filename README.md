> Weixin message handling


## Install

```sh
$ npm install --save node-weixin-message
```


## Usage

```js
var nodeWeixinMessage = require('node-weixin-message');

nodeWeixinMessage('Rainbow');
```

```sh
$ npm install --global node-weixin-message
$ node-weixin-message --help
```
